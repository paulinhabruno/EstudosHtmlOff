Pseudo classes

  • As pseudo classes são indicadas no CSS com : (dois pontos) e devem estar relacionadas a um elemento ou a uma class.
  • Está relacionada ao estado de um determinado elemento.
  • Tipos de estado:
    1. Active
    2. Checked
    3. Hover = quando passa o mouse por cima.
    4. Empty
    5. Entre outras.
