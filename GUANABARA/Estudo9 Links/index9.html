Usando links externos

Você pode acessar o meu repositório público no Github.

Para que o site seja direcionado para outra janela, deve-se inserir o TARGET=_BLANK e o REL="EXTERNAL" dentro do endereço.

Abaixo sem os comandos, abrirá na mesma aba.

Você também pode acessar o site do Curso em Vídeo do Guanabara.

Usando link interno.

Para usar o link interno cria-se outro HTML dentro da mesma página com outro nome, mas com a mesma extensão.


Essa é a página principal, mas acessar a segunda página clique aqui.
